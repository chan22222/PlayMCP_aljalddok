# AI비서 알잘똑

알아서 잘 딱 깔끔하게 도와주는 AI 비서! Gemini AI를 활용하여 일정 관리, 맛집 검색, 그룹 채팅 요약, 할 일 추출 등 다양한 기능을 제공합니다.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set `GEMINI_API_KEY` in `.env.local`:
   ```
   GEMINI_API_KEY=your_api_key_here
   ```
3. Run the app:
   `npm run dev`
